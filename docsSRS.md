# Software Requirements Specification (SRS)

## 1. Review of Given Requirements

### Requirement 1:
"The system should be user-friendly."
✅ The system shall allow a new user to complete registration within 3 minutes without external assistance.

### Requirement 2:
"The system shall store student records safely."
✅ The system shall encrypt all student records using AES-256 encryption and allow access only to authenticated users.

### Requirement 3:
"The system should load quickly."
✅ The system shall load the dashboard page within 2 seconds under normal network conditions (minimum 10 Mbps connection).

### Requirement 4:
"The system shall allow users to register, login, and manage their profile."
✅ The system shall allow users to register using a valid email and password.
✅ The system shall allow registered users to log in using valid credentials.
✅ The system shall allow users to update their profile information.

### Requirement 5:
"The system shall send notifications."
✅ The system shall send an email notification to users within 1 minute after successful registration.
