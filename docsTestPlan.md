# Test Plan

## 1. Introduction
This document describes the testing strategy for the Student Management System.

## 2. Test Scope
- User Registration
- Login
- Profile Management
- Student Record Storage
- Notifications

## 3. Test Types
- Functional Testing
- Security Testing
- Performance Testing

## 4. Sample Test Cases
### TC-01: User Registration
Input: Valid email and password  
Expected Result: User account created successfully  

### TC-02: Login
Input: Valid credentials  
Expected Result: User successfully logged in  

### TC-03: Dashboard Load Time
Condition: Normal 10 Mbps internet  
Expected Result: Dashboard loads within 2 seconds  

### TC-04: Data Security
Condition: Attempt unauthorized access  
Expected Result: Access denied  

### TC-05: Notification
Condition: Successful registration  
Expected Result: Email notification sent within 1 minute
